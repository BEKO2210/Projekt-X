# Projekt-X — Sovereign AI Business Companion

> **Status:** Konzept- und Architekturphase (Phase 0 der Roadmap).
> **Kein Code. Keine App. Keine UI. Keine Agentenautomation. Keine Integrationen.**
> Dieses Repository enthält ausschließlich Dokumentation, Steuerdateien und Arbeitsregeln.

## Was dieses Projekt ist

Projekt-X (Arbeitstitel: **Sovereign AI Business Companion**) ist ein Konzept für ein KI-basiertes Begleitsystem für:

- berufliche Orientierung
- berufliche Neuorientierung
- Prüfung und Aufbau von Selbstständigkeit
- strukturierte Projektbegleitung
- Aufgaben- und Workflowführung
- Erzeugung von Dokumententwürfen mit Status
- sichtbarer Quellenstatus und Fachprüfungsstatus
- Risikoübersicht
- Datenschutzmodus mit Nutzerkontrolle
- Lernpfade für Kompetenzaufbau

Das System ist als modulares, agentisches Konzept geplant, in dem spezialisierte Agenten unter einem CEO-Orchestrator zusammenarbeiten. Die Verfassung des Projekts liegt in `KI_COMPANION_BLUEPRINT.md`.

## Was dieses Projekt ausdrücklich nicht ist

- kein Ersatz für Rechts-, Steuer-, Finanz- oder medizinische Beratung
- keine Garantie für Fördermittel
- kein automatischer Behörden- oder Kundenkommunikator
- keine Buchhaltungs-, CRM- oder Zahlungsplattform
- keine produktionsreife Software in diesem Stadium
