- Dritte Inhalte (Zitate, Bilder, Muster) werden im Text entsprechend ausgewiesen, sobald vorhanden.

Eine abschließende Lizenzdatei (z. B. MIT, Apache-2.0, GPL, AGPL, CC-Varianten, kommerziell) wird diese Datei in einem späteren Run ersetzen.

## 2. Zu entscheiden

Der Lizenz-ADR wird mindestens behandeln:

- Zielgruppe und Nutzungsziel

- Markenrechte
- Haftungsausschluss
- Trennung Code / Inhalte / Prompts / Vorlagen

Bis zur Entscheidung gilt der oben genannte **Vorbehalt**.
