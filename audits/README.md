# audits/

## Zweck

**Prüfberichte**: interne und externe Audits zu Datenschutz, Sicherheit, Agentenverhalten, Quellen, Compliance.

## Konvention (geplant)

- Dateiname: `YYYY-MM-DD-<audit-typ>.md` (z. B. `2026-XX-XX-datenschutz-review.md`).
- Pflichtteile: Scope, Prüfer, Methode, Befunde, Risiken, Empfehlungen, Status.

## Stand Run 1

Leer. Audit-Routinen werden in späteren Phasen etabliert (insbesondere Phase 8).
