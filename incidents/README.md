# incidents/

## Zweck

**Incident-Berichte**: jeder Sicherheits- oder Datenschutzvorfall wird hier dokumentiert.

## Konvention (geplant)

- Dateiname: `YYYY-MM-DD-<kurzbezeichnung>.md`.
- Pflichtteile: Zeit, Auslöser, betroffene Daten/Personen, Sofortmaßnahme, Ursache, Post-Mortem, Lernpunkte.
- Lernpunkte, die systemisch wirken, werden in `CLAUDE.md` / betroffene Dokumente übernommen.
